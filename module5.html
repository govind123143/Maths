3
Explain Singular Value Decomposition (SVD).
CO 5L33 M

SVD factorises any real \(m\times n\) matrix A into three matrices:

\[A = U\Sigma V^T\]
  • U — \(m\times m\) orthogonal matrix (left singular vectors = eigenvectors of \(AA^T\))
  • Σ — \(m\times n\) diagonal matrix with singular values \(\sigma_1\geq\sigma_2\geq\ldots\geq0\)
  • V — \(n\times n\) orthogonal matrix (right singular vectors = eigenvectors of \(A^TA\))

Singular values: \(\sigma_i=\sqrt{\lambda_i(A^TA)}\). SVD always exists for any matrix.

4
Mention any three applications of SVD.
CO 5L33 M
  1. Image Compression: Approximate a matrix with a lower-rank matrix by keeping only the top-k singular values, reducing storage while preserving essential information.
  2. PCA / Dimensionality Reduction: SVD is the backbone of Principal Component Analysis — finds directions of maximum variance to reduce feature dimensions.
  3. Solving Linear Systems (Pseudoinverse): \(A^+=V\Sigma^+U^T\) provides the least-squares solution to overdetermined systems \(Ax=b\).
  4. Recommender Systems: Collaborative filtering (Netflix, Amazon) uses SVD to discover latent patterns in user-item rating matrices.
5 (a)
State the properties of vectors.
CO 5L33 M
  • Commutativity: \(\vec{a}+\vec{b}=\vec{b}+\vec{a}\)
  • Associativity: \((\vec{a}+\vec{b})+\vec{c}=\vec{a}+(\vec{b}+\vec{c})\)
  • Zero vector: \(\vec{a}+\vec{0}=\vec{a}\)
  • Scalar multiplication: \(k(\vec{a}+\vec{b})=k\vec{a}+k\vec{b}\)
  • Dot product: \(\vec{a}\cdot\vec{b}=|\vec{a}||\vec{b}|\cos\theta\) (commutative)
  • Cross product: \(\vec{a}\times\vec{b}=-(\vec{b}\times\vec{a})\) (anti-commutative)
  • Orthogonality: \(\vec{a}\perp\vec{b}\iff\vec{a}\cdot\vec{b}=0\)

6
Solve by LU decomposition: 3x+2y+7z=4, 2x+3y+z=5, 3x+4y+z=7
CO 5L314 M
\[A=\begin{pmatrix}3&2&7\\2&3&1\\3&4&1\end{pmatrix},\quad b=\begin{pmatrix}4\\5\\7\end{pmatrix}\]

Step 1: LU Decomposition (A = LU)

  1. U[Row 1] = [3, 2, 7]
  2. \(l_{21}=2/3,\;l_{31}=1\). R2−(2/3)R1→[0, 5/3, −11/3]. R3−R1→[0, 2, −6]
  3. \(l_{32}=2/(5/3)=6/5\). R3−(6/5)R2→[0, 0, −8/5]
  4. \(L=\begin{pmatrix}1&0&0\\2/3&1&0\\1&6/5&1\end{pmatrix},\quad U=\begin{pmatrix}3&2&7\\0&5/3&-11/3\\0&0&-8/5\end{pmatrix}\)

Step 2: Forward Substitution Ly = b

  1. \(y_1=4\)
  2. \((2/3)(4)+y_2=5\implies y_2=5-8/3=7/3\)
  3. \((1)(4)+(6/5)(7/3)+y_3=7\implies y_3=7-4-14/5=1/5\)

Step 3: Back Substitution Ux = y

  1. \((-8/5)z=1/5\implies z=-1/8\)
  2. \((5/3)y+(-11/3)(-1/8)=7/3\implies y=9/8\)
  3. \(3x+2(9/8)+7(-1/8)=4\implies 3x=21/8\implies x=7/8\)
x = 7/8,  y = 9/8,  z = −1/8

Solve by LU decomposition: 2x+y+z=2, x+3y+2z=2, 3x+y+z=2
CO 5L314 M
\[A=\begin{pmatrix}2&1&1\\1&3&2\\3&1&1\end{pmatrix},\quad b=\begin{pmatrix}2\\2\\2\end{pmatrix}\]

LU Decomposition

  1. \(l_{21}=1/2,\;l_{31}=3/2\)
  2. R2−(1/2)R1→[0, 5/2, 3/2]. R3−(3/2)R1→[0, −1/2, −1/2]
  3. \(l_{32}=(−1/2)/(5/2)=−1/5\). R3−(−1/5)R2→[0, 0, −1/5]
  4. \(U=\begin{pmatrix}2&1&1\\0&5/2&3/2\\0&0&-1/5\end{pmatrix}\)

Forward Substitution: Ly = b

  1. \(y_1=2\)
  2. \((1/2)(2)+y_2=2\implies y_2=1\)
  3. \((3/2)(2)+(−1/5)(1)+y_3=2\implies y_3=2−3+1/5=−4/5\)

Back Substitution: Ux = y

  1. \((-1/5)z=-4/5\implies z=4\)
  2. \((5/2)y+(3/2)(4)=1\implies y=-2\)
  3. \(2x+(-2)+4=2\implies x=0\)
x = 0,  y = −2,  z = 4

Solve by LU decomposition: 4x+y+z=4, x+4y−2z=4, 3x+2y−z=6

\[A=\begin{pmatrix}4&1&1\\1&4&-2\\3&2&-1\end{pmatrix},\quad b=\begin{pmatrix}4\\4\\6\end{pmatrix}\]

LU Decomposition

  1. \(l_{21}=1/4,\;l_{31}=3/4\)
  2. R2−(1/4)R1→[0, 15/4, −9/4]. R3−(3/4)R1→[0, 5/4, −7/4]

  1. \(-z=2\implies z=-2\)
  2. \((15/4)y+(−9/4)(−2)=3\implies(15/4)y=3−18/4=−1.5\implies y=-0.4\)
  3. \(4x+(−0.4)+(−2)=4\implies 4x=6.4\implies x=1.6\)
x = 1.6,  y = −0.4,  z = −2
9
Find SVD for A = [[1,0,1],[−1,1,0]]
CO 5L314 M
  1. Compute \(AA^T\) (2×2): \[AA^T=\begin{pmatrix}1&0&1\\-1&1&0\end{pmatrix}\begin{pmatrix}1&-1\\0&1\\1&0\end{pmatrix}=\begin{pmatrix}2&-1\\-1&2\end{pmatrix}\]
  2. Eigenvalues of \(AA^T\): \(\lambda^2-4\lambda+3=0\implies(\lambda-3)(\lambda-1)=0\implies\lambda_1=3,\;\lambda_2=1\)
  3. Singular values: \(\sigma_1=\sqrt{3},\;\sigma_2=1\)
  4. \(\Sigma=\begin{pmatrix}\sqrt{3}&0&0\\0&1&0\end{pmatrix}\)
  5. For λ=3: eigenvector of \(AA^T\) → \(u_1=\frac{1}{\sqrt{2}}\begin{pmatrix}1\\-1\end{pmatrix}\)
  6. For λ=1: \(u_2=\frac{1}{\sqrt{2}}\begin{pmatrix}1\\1\end{pmatrix}\)
  7. Right singular vectors: \(v_1=\frac{1}{\sigma_1}A^Tu_1=\frac{1}{\sqrt{6}}\begin{pmatrix}2\\-1\\1\end{pmatrix},\;v_2=\frac{1}{\sqrt{2}}\begin{pmatrix}0\\1\\1\end{pmatrix}\)
  8. \(v_3\) = null space of A: \(v_3=\frac{1}{\sqrt{3}}\begin{pmatrix}-1\\1\\1\end{pmatrix}\)
\[U=\frac{1}{\sqrt{2}}\begin{pmatrix}1&1\\-1&1\end{pmatrix},\quad\Sigma=\begin{pmatrix}\sqrt{3}&0&0\\0&1&0\end{pmatrix},\quad V=\begin{pmatrix}\frac{2}{\sqrt{6}}&0&\frac{-1}{\sqrt{3}}\\\frac{-1}{\sqrt{6}}&\frac{1}{\sqrt{2}}&\frac{1}{\sqrt{3}}\\\frac{1}{\sqrt{6}}&\frac{1}{\sqrt{2}}&\frac{1}{\sqrt{3}}\end{pmatrix}\]
Singular values: σ₁ = √3, σ₂ = 1 (σ₃ = 0)
10
Find SVD for A = [[1,1,0],[0,0,1]]
CO 5L314 M
  1. Compute \(AA^T\) (2×2): \[AA^T=\begin{pmatrix}1&1&0\\0&0&1\end{pmatrix}\begin{pmatrix}1&0\\1&0\\0&1\end{pmatrix}=\begin{pmatrix}2&0\\0&1\end{pmatrix}\]
  2. Eigenvalues: \(\lambda_1=2,\;\lambda_2=1\)
  3. Singular values: \(\sigma_1=\sqrt{2},\;\sigma_2=1\)
  4. \(\Sigma=\begin{pmatrix}\sqrt{2}&0&0\\0&1&0\end{pmatrix}\)
  5. Left singular vectors: \(u_1=(1,0)^T\) for λ=2; \(u_2=(0,1)^T\) for λ=1
  6. Right singular vectors: \(v_1=\frac{1}{\sqrt{2}}A^Tu_1=\frac{1}{\sqrt{2}}(1,1,0)^T\)
  7. \(v_2=A^Tu_2=(0,0,1)^T\)
  8. \(v_3\) = orthogonal complement: \(v_3=\frac{1}{\sqrt{2}}(1,-1,0)^T\)
\[U=I_2=\begin{pmatrix}1&0\\0&1\end{pmatrix},\quad\Sigma=\begin{pmatrix}\sqrt{2}&0&0\\0&1&0\end{pmatrix}\] \[V=\begin{pmatrix}\frac{1}{\sqrt{2}}&0&\frac{1}{\sqrt{2}}\\\frac{1}{\sqrt{2}}&0&\frac{-1}{\sqrt{2}}\\0&1&0\end{pmatrix}\]
Singular values: σ₁ = √2, σ₂ = 1
\(A = U\Sigma V^T\) decomposition complete.
